Meu portifólio - Ana Beatriz Magarotto
Um dos meus primeiros projetos como desenvolvedora foi a criação, do que na época, era apenas um modelo de como seria meu portifólio pessoal.
Após alguns anos, resolvi revisitar esse projeto e atualizá-lo, além de claro, colocar um pouco mais de toque pessoal.
Muito bom retormar esse projeto e perceber o quanto evoluí e o quanto ainda há para aprender.
Espero atualizá-lo em breve com minha futura vaga de estágio na área de desenvolvimento!
